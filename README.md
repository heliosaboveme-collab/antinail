# antinail

【概要】
「世の中の人がより生きやすくなるものを作る」という信念のもと開発している、YouTubeの誹謗中傷コメント自動取得・判定ツールです。Google AI Studio（Gemini API）とYouTube APIを連携させ、AIと対話しながらロジックを組み立てる「AI共創型」のオリジナルプロジェクトとして制作しています。

【工夫したポイント】
Google AI Studioによるコメントの判定
単なる単語一致ではなく、文脈から攻撃性を判断するプロンプトを練り込み、巧妙な言い回しの誹謗中傷も検知できる仕組みを構築しました。
差分更新による処理の高速化
取得済みコメントIDとの照合ロジックを自作し、新着分のみを抽出。API呼び出しを最小限に抑え、スピーディーな更新を可能にしました。
運用コストの徹底削減
AIの文字単位の従量課金対策として、危険度の低い判定には「C」と一文字のみ出力させるプロンプト設計を行い、トークン消費を大幅にカットしています。

【改善点】
現在はテキスト判定が主軸ですが、今後は各コメントの個別URLから自動でスクリーンショットを撮影・保存する機能の追加を予定しています。これにより、bot判定などの制限を回避しつつ、法的証拠としての信頼性をより強化することを目指しています。

【実行方法】
本ツールはGoogle AI Studioおよびスプレッドシート環境で動作するため、GitHub上ではソースコードと構築ロジックの公開のみとなります。
